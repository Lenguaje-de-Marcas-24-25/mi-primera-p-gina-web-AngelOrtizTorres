Respuestas sobre HTML y XHTML

¿Qué es el lenguaje HTML?

HTML (HyperText Markup Language) es el lenguaje estándar de marcado utilizado para crear y estructurar contenido en la web. Permite definir la estructura básica de una página web mediante elementos como encabezados, párrafos, enlaces, imágenes, y otros elementos multimedia.

Relación de Tim Berners-Lee con HTML

Tim Berners-Lee, un científico británico, es el creador del lenguaje HTML. En 1989, desarrolló el primer prototipo de HTML como parte de su proyecto para crear la World Wide Web, donde estableció un sistema para enlazar documentos a través de hipertextos, permitiendo la navegación entre páginas web.

Distintas versiones del lenguaje HTML

Las versiones del lenguaje HTML han evolucionado desde HTML 1.0 hasta HTML5, con mejoras en la estructura, nuevas etiquetas y soporte para elementos multimedia. Las versiones más importantes incluyen HTML 2.0, HTML 3.2, HTML 4.01 y la versión actual, HTML5, que introduce capacidades avanzadas para aplicaciones interactivas y multimedia.

¿Qué es la W3C?

La W3C (World Wide Web Consortium) es una organización internacional que se encarga de desarrollar estándares para la web, incluyendo HTML y CSS. Su objetivo es asegurar el crecimiento y accesibilidad de la web mediante el establecimiento de directrices y normativas que promuevan la interoperabilidad y accesibilidad.

¿Qué es el WHATWG?

El WHATWG (Web Hypertext Application Technology Working Group) es un grupo de trabajo que desarrolla y mantiene el HTML Living Standard. Fundado en 2004 por empresas de tecnología como Apple, Mozilla y Opera, el WHATWG busca mejorar y actualizar continuamente los estándares de HTML con un enfoque en las necesidades modernas de la web.

¿Qué es un Living Standard?

Un "Living Standard" es una especificación que está en constante evolución y actualización, en lugar de tener versiones fijas. Esto permite que el estándar se adapte a las necesidades cambiantes de los usuarios y a los avances tecnológicos sin tener que esperar a revisiones o versiones futuras.

¿Qué es el lenguaje XHTML?

XHTML (Extensible Hypertext Markup Language) es una reformulación de HTML bajo la sintaxis estricta de XML (eXtensible Markup Language). XHTML se creó para hacer que el marcado de las páginas web fuera más estricto y predecible, favoreciendo la interoperabilidad y el cumplimiento de estándares.

Distintas versiones del lenguaje XHTML

Las versiones de XHTML incluyen XHTML 1.0, que fue la primera versión en basarse en HTML 4.01 con sintaxis XML, y XHTML 1.1, que introdujo mejoras adicionales en la estructura y modularización del lenguaje. XHTML 2.0 fue un intento posterior que nunca fue adoptado ampliamente, y el proyecto se detuvo en favor de HTML5.

Diferencias sintácticas y estructurales entre XHTML y HTML

XHTML se basa en la sintaxis de XML, lo que significa que es más estricto que HTML. Algunas de las principales diferencias son:

  • Los elementos deben estar correctamente anidados y cerrados, incluso las etiquetas vacías como <br /> en lugar de <br>.
  • Los nombres de las etiquetas y atributos deben estar en minúsculas.
  • Todos los atributos deben tener un valor; por ejemplo, checked="checked" en lugar de solo checked.
  • El uso de DTD (Document Type Definition) para declarar la versión del documento en XHTML.
