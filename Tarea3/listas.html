Listas

A continuación nos encontramos un ejemplo de cómo utilizar las listas.

  1. W3C (World Wide Web Consortium):
    Organismo internacional que crea estándares para la web, como HTML y CSS.
  2. WHATWG (Web Hypertext Application Technology Working Group):
    Grupo que desarrolla estándares web, responsable de HTML5.
  3. HTML (HyperText Markup Language):
    Lenguaje principal para estructurar contenido en la web.
  4. XHTML (Extensible HyperText Markup Language):
    Versión de HTML que sigue estrictas reglas de XML.
  5. DTD (Document Type Definition):
    Define la estructura y las reglas de documentos XML y HTML.
  6. APIs:
    1. Drag and Drop:
      Permite arrastrar y soltar elementos en la interfaz web.
    2. Geolocalización:
      Permite a las aplicaciones obtener la ubicación del usuario.
    3. Almacenamiento Local (Local Storage):
      Guarda datos del usuario en el navegador para su uso offline.
    4. App Cache:
      Carga aplicaciones web sin conexión mediante el almacenamiento de recursos en caché (obsoleto en favor de Service Workers).
    5. WebWorkers:
      Permite ejecutar tareas en segundo plano en el navegador sin bloquear la interfaz.
  7. Editor WYSIWYG (What You See Is What You Get):
    Permite crear contenido visualmente, sin necesidad de escribir el código.
  8. Navegadores:
    1. Mozilla Firefox:
      Navegador de código abierto desarrollado por Mozilla.
    2. Google Chrome:
      Navegador rápido y popular desarrollado por Google.
    3. Safari:
      Navegador web de Apple, optimizado para dispositivos de Apple.
    4. Opera:
      Navegador rápido y versátil con características avanzadas, desarrollado por Opera Software.
